Announcing a collaborative platform for all

You are welcome onboard to collaborate and contribute, brainstorm ideas, design, plan, document, and code. Create apps, build web platforms, as well as work with tools that developers use - Slack, Git, and many more...


AbyssalTech is made for the purpose of sharing knowledge, experiences, and anything that is tech related. We want to extend that experience further, allowing you to get insights directly from the developers and get hands-on experience with building stuff.

  • Frontend and Backend Development
